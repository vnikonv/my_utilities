### Things To Consider:

* My utilities are unsafe. Use them at your own risk.
* Some of the utilities enable batch processing for tasks that are executed by third-party programs (such as ffmpeg or 7z).
* The scripts will not work correctly, unless the relevant program is already installed.
* As for Python scripts, you need Python installed to execute them.
* Also, note that VBS scripts are written specifically for Windows with the assumption that your system has a VBS script interpreter.
